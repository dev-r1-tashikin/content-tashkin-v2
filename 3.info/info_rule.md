# 网站 URL 目录规则
* `url`：网站 URL
* `dir`：网站目录
* `path`：网站路径
* `domain`：网站域名
* `subdomain`：网站子域名
* `protocol`：网站协议（http/https）
* `port`：网站端口
* `query`：网站查询参数
* `fragment`：网站片段
* `params`：网站参数
* `headers`：网站头部
## 层级规则
* 大分类主题页： /大分类
* 子分类主题页： /大分类/子分类
* 产品页： /大分类/产品名称
* 所有产品页：/大分类/products
